# Hi 👋, I'm Maple

## 🚀 關於我
我是一名熱愛程式開發的工程師，專注於**金融科技**和**醫療科技**領域的創新解決方案。我喜歡創造讓數據更易於理解和使用的工具。

## 🎯 主要專案

### 📊 上市公司財報分析系統
一個全面的上市公司財務報表分析系統，提供深入的數據洞察和視覺化圖表，協助投資決策分析。

### 💰 個人記帳系統
直觀易用的個人財務管理工具，幫助使用者追蹤支出、管理預算，並視覺化消費模式。

### 🏥 DICOM 檢視與標記工具
專業的醫學影像檢視與標註工具，支援 DICOM 格式，協助醫療專業人員進行診斷工作流程。
